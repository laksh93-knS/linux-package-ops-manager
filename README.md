# Linux Package Operations Manager 

## Description
This project is a simple Bash script that helps manage Linux packages.

It provides:
- System update option
- Package installation
- Package removal
- Simple menu-based interface

---

## Prerequisites
- Linux system (Ubuntu/Debian recommended)
- Sudo access
- Internet connection

---

## Installation

Navigate folder:
cd linux-package-ops-manager

Make script executable:
chmod +x package_manager_helper.sh

---

## Usage

Run script:
./package_manager_helper.sh

---

## Output

Menu will appear:
1. Update system
2. Install package
3. Remove package
